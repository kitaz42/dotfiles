# 🐧 kitaz42 dotfiles

каааайф оч круто но для амд не подойдет, только для нвидиа 

- **OS:** Arch Linux (NO ARCHINSTALL) with repositories and the Cache OS kernel
- **WM:** Hyprland
- **Shell:** Zsh (+ syntax-highlighting & autosuggestions)
- **Terminal:** Kitty / Alacritty

## 🚀 Быстрый старт

```bash
git clone https://github.com/kitaz42/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```

> **Warning**: Скрипт заменяет существующие конфиги в `~/.config/hypr` символическими ссылками.
