## 🚀 Быстрый старт

```bash
git clone https://github.com/kitaz42/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```

> **Warning**: Скрипт заменяет существующие конфиги в `~/.config/hypr` символическими ссылками. Также в hyprland.conf 74.97 на свою
