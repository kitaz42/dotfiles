# 🐧 kitaz42 dotfiles
 
Внимание: Эти конфиги настроены под мои комплектующие. Используйте на свой страх и риск. Я не несу ответственности за сломанные системы или взорванные мониторы.
- **OS:** - Arch Linux (NO ARCHINSTALL) with repositories and the CacheOS kernel
- **WM:** Hyprland
- **PC components:** NVIDIA Geforce RTX 2060 SUPER/Ryzen 5 3600/24GB RAM.

- **Shell:** Zsh (+ syntax-highlighting & autosuggestions)
- **Terminal:** Kitty / Alacritty

![Setup](screenshot.png)

## 🚀 Быстрый старт

```bash
git clone https://github.com/kitaz42/dotfiles.git
chmod +x install.sh
~/dotfiles/install.sh
```

> **Warning**: Скрипт заменяет существующие конфиги в `~/.config/hypr` символическими ссылками. Также в hyprland.conf 74.97 на свою
